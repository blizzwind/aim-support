# aim-support
A script supports aiming using convolutional neural networks.

### dependencies
- pywin32
- mss
- gluoncv

### usage
```
git clone https://github.com/blizzwind/aim-support
cd aim-support
python script.py
```

### notice
OS should be Windows<br>
script may crash<br>
process is slow due to reading and writing
